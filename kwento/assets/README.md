# Kwento Mode Assets

This folder contains the game assets for Kwento Mode.

## Required Assets (Download from OpenGameArt.org)

### 1. LPC Base Assets
- **URL:** https://opengameart.org/content/lpc-base-assets
- **License:** CC-BY-SA 3.0 / GPL 3.0
- **Download:** Download the ZIP file
- **Extract to:** kwento/assets/tiles/lpc_base.png (or keep as is and update path in code)
- **Contains:** 32x32 tiles for grass, dirt, stone, water, walls, doors, windows

### 2. LPC Medieval Town
- **URL:** https://opengameart.org/content/lpc-medieval-town
- **License:** CC-BY-SA 3.0 / GPL 3.0
- **Download:** Download the ZIP file
- **Extract to:** kwento/assets/tiles/lpc_town.png
- **Contains:** Town buildings, cobblestone streets, market stalls

### 3. LPC Forest
- **URL:** https://opengameart.org/content/lpc-forest
- **License:** CC-BY-SA 3.0 / GPL 3.0
- **Download:** Download the ZIP file
- **Extract to:** kwento/assets/tiles/lpc_forest.png
- **Contains:** Trees, bushes, forest floor

### 4. LPC Characters
- **URL:** https://opengameart.org/content/lpc-character-base
- **License:** CC-BY-SA 3.0 / GPL 3.0
- **Download:** Download the ZIP file
- **Extract to:** kwento/assets/sprites/
- **Contains:** Character sprites with walking animations

## Asset Organization

```
kwento/assets/
├── tiles/
│   ├── lpc_base.png          # Base tiles (grass, dirt, stone)
│   ├── lpc_town.png          # Town buildings
│   └── lpc_forest.png        # Forest tiles
├── sprites/
│   ├── player.png            # Player sprite (from LPC Characters)
│   ├── bonifacio.png         # NPC sprite
│   ├── soldier.png           # Enemy sprite
│   └── merchant.png          # NPC sprite
└── audio/                    # Optional: sound effects
```

## Instructions

1. Download each asset from the URLs above
2. Extract the ZIP files
3. Place the PNG files in the appropriate folders
4. For Phase 1-2, use tiles as-is (no recoloring)
5. Recoloring for Spanish colonial aesthetic will be done in Phase 4

## Placeholder Assets

For testing purposes, you can use simple colored rectangles as placeholders:
- 32x32 colored squares for tiles
- Simple shapes for sprites

This will allow development to proceed while assets are being prepared.
